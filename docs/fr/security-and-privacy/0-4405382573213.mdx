# Comment demander des journaux d’audit?

Si vous devez demander des journaux d’audit, veuillez contacter votre représentant des ventes Philips.

Dans votre courriel, veuillez indiquer :

- raison de la demande de journaux d'audit
- la période de journaux d'audit (avec le fuseau horaire où l'application est utilisée)
- nom de compte d’utilisateur ou de machine
