# Aperçu

## Introduction

Bienvenue sur Reacts ! Vous trouverez ici des informations complètes sur l'intégration à la plateforme Reacts. Nous avons essayé de rendre cette documentation conviviale et remplie d'exemples

Si vous envisagez d'utiliser notre API en production, consultez notre [Avis de confidentialité](https://reacts.com/en/legal/privacy#openInNewTab) et nos [Conditions d'utilisation](https://reacts.com/en/legal/terms-and-conditions#openInNewTab).

---

## Protocoles API
L'API Reacts utilise des requêtes HTTP POST/GET/PUT pour communiquer et des codes de réponse HTTP pour indiquer l'état et les erreurs. Toutes les réponses sont au format JSON standard. L'API Reacts est servie sur HTTPS TLS v1.2+ pour garantir la confidentialité des données ; HTTP et HTTPS avec des versions TLS inférieures à 1.2 ne sont pas pris en charge. Toutes les requêtes doivent inclure un type de contenu d'application/json et le corps doit être un JSON valide.

## Gestion des versions

Nous publions périodiquement de nouvelles versions de notre API. Nous faisons de notre mieux pour minimiser les changements avec rupture. Dans le cas où un changement de rupture est introduit, vous en serez informé.
