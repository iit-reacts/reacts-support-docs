**Cryptage local**

Pour une sécurité supplémentaire, votre contenu Reacts peut maintenant être chiffré sur votre appareil. Si vous voulez cette fonctionnalité, cliquez sur "Je veux un cryptage local" durant la mise à jour. Ceci vous permettra d’installer le pilote EldoS Corporation qui est requis pour activer cette fonctionnalité Reacts.  
  
**Identification des licences « pro » et « invité » dans le panneau de contacts**

Pour vous permettre de mieux identifier vos contacts qui ont une licence de type « Pro » ou de type « Invité », nous avons ajouté un descriptif visuel dans le panneau de contacts. Les utilisateurs qui ont une licence de type «Standard » n’ont pas de descriptif visuel spécifique.
