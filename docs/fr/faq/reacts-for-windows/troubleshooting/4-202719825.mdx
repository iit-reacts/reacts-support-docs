title: Le pare-feu de Windows affiche un message lorsque certaines fonctionnalités de Reacts sont utilisées pour la première fois
---
Lorsque vous utilisez certaines fonctionnalités de Reacts pour la première fois (p. ex. : partager l’application/le bureau), le pare-feu de Windows les bloquera et le message suivant s’affichera:

 ![](@assets/attachments/windows_firewall_message.png)

Il vous suffit de sélectionner **_Autoriser l’accès_** pour continuer
