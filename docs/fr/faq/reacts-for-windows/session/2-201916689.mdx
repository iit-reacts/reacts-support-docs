_Le panneau d’actions proposera différentes options si vous créez/modifiez une session, si vous essayez d’établir une connexion avec un contact ou prenez part à une session en direct._

_![](@assets/attachments/actions1.png)_

 Ajouter une nouvelle scène à votre présentation

**Vous pouvez consulter le tutoriel vidéo Ajouter une nouvelle scène** ([https://www.youtube.com/watch?v=-C8T2pEwhn8&t=1s](https://www.youtube.com/watch?v=-C8T2pEwhn8&t=1s))

**_Établir une connexion avec un autre utilisateur Reacts à partir de votre liste de contacts_**

![mceclip0.png](@assets/attachments/8429508891677.png)

Après avoir sélectionné l’option **_Ajouter à l'appel,_** vous accèderez à une liste de vos contacts. Il vous suffit de choisir l’un de vos contacts dans la liste et d’établir une connexion.

![mceclip1.png](@assets/attachments/8429510624925.png)

**Vous pouvez consulter le tutoriel vidéo "Établir une connexion avec un contact" (à partir de l'interface principale de la session) ([https://www.youtube.com/watch?v=KjuCd2G0HGI](https://www.youtube.com/watch?v=KjuCd2G0HGI))**
