title: Comment puis-je changer le format de la scène (4/3, 16/9)?
---
Lorsque vous êtes en mode session, vous pouvez changer le format d'une scène. Par défaut chaque scène est au format 16/9 mais vous pouvez toutefois choisir le format 4/3.

Dans la _**fenêtre de la scène**_, choisir l'option **_Format_** sur  _**la barre de menu**__._

![format_scene.png](@assets/attachments/format_scene.png)

_**Veuillez noter que le Format doit être changé avant tout ajout d'arrière-plan ou d'élément superposé autrement votre arrière-plan sera réorganisé et tous les éléments mis en superposition enlevés.**_
