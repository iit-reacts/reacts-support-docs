**Les options suivantes s’affichent sur la barre de menu**
----------------------------------------------------------

 ****Saisir un instantané ou enregistrer une courte vidéo :****

**![Snapshots_FR.png](@assets/attachments/Snapshots_FR.png)**

Vous permet de prendre une photo de n’importe quel élément sur une toile. Tous les instantanés saisis au cours d’une session s’affichent dans la galerie d’instantanés et sont stockés dans la **bibliothèque d’instantanés** sous un dossier portant la date à laquelle l’instantané a été saisi.

**Vous pouvez consulter le tutoriel vidéo [Instantanés (photo: https://www.youtube.com/watch?v=vh8oougq8t8, video : https://www.youtube.com/watch?v=NupIc5ektvw)](https://www.youtube.com/watch?v=vh8oougq8t8)**

**Gestion des couches:**

**![](@assets/attachments/session_interface-3.png)**

Si vous avez **plusieurs éléments** sur une même scène, le **menu Couches** vous aidera à gérer ces éléments.

![](@assets/attachments/session_interface-4.png)

**Verrouiller ou déverrouiller votre arrière-plan:**

**![](@assets/attachments/session_interface-5.png)**

 Vous pouvez insérer un objet ou une entrée sur une toile en tant qu’**arrière-plan** ou en tant qu’**élément**.

Pour passer facilement de la gestion de l'**Arrière-plan** à la gestion des **Éléments**, vous pouvez **verrouiller/déverrouiller** votre arrière-plan.

 **Vous pouvez consulter le tutoriel vidéo [Arrière-plan https://www.youtube.com/watch?v=0GzVNPuSqPo](https://www.youtube.com/watch?v=0GzVNPuSqPo)**

**Modifier le rapport de dimensions de la caméra:**

**![](@assets/attachments/session_interface-7.png)**

Par défaut, le rapport est établi à la valeur la plus appropriée, mais vous pouvez le modifier à partir de la **barre de menu**.

 _Veuillez noter que le rapport doit être modifié avant l’ajout d’éléments. Autrement, les éléments en place seront supprimés._

**Vous pouvez consulter le tutoriel vidéo [Ajuster le format d'une fenêtre de scène (ratios recommandés: 4:3, 16:9) (https://www.youtube.com/watch?v=Gw81GgfeYV8)](https://www.youtube.com/watch?v=Gw81GgfeYV8)**
