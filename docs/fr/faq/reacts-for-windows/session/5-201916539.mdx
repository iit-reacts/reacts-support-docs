title: Comment accéder à mes sessions sauvegardées et les gérer?
---
Pour trouver une session enregistrée au préalable dans le menu principal, cliquez sur : **Sessions --> Commencer --> Sessions sauvegardées**

À partir de cet emplacement, vous pouvez ouvrir, gérer, modifier, ou supprimer des sessions sauvegardées au préalable ou en supprimer pour organiser vos sessions, ou encore les déplacer d’un dossier à l’autre:

![](@assets/attachments/Access_and_manage_saved_sessions-1.png)

Faites un clic droit sur la session de votre choix pour l’ouvrir, la supprimer ou la renommer:

![](@assets/attachments/Access_and_manage_saved_sessions-2.png)
