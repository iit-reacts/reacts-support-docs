title: Comment ajuster les paramètres audio/vidéo?
---
**Vous pouvez consulter le tutoriel vidéo [Entrées vidéo (https://www.youtube.com/watch?v=2gjRyiFrKJ4)](https://www.youtube.com/watch?v=2gjRyiFrKJ4)**

Vous pouvez ajuster vos **_paramètres Audio/Vidéo_** directement dans l’interface de la session. À partir du **_panneau d’actions_** accédez aux **paramètres**, puis sélectionnez **_Audio/Vidéo_**.

![FR_In_session.png](@assets/attachments/8274541870493.png)

**_Configurez ou ajustez ce qui suit :_**

![](@assets/attachments/one.png) Microphone

![](@assets/attachments/two.png) Écouteurs/sorties

![](@assets/attachments/three.png) Caméra (pour des paramètres de caméra supplémentaires, consultez le document "Paramètres vidéo/Propriétés de la caméra":

![](@assets/attachments/settings4.png)

Vous pouvez également accéder aux paramètres audio/vidéo à partir du menu principal. Pour plus d'information veuillez consulter le document "Comment modifier les paramètres et les renseignements de compte dans Reacts?"
