Description
===========

Le _**Système de mises à jour silencieuses**_ est un mécanisme de Reacts destiné à maintenir l'application à jour. Ce système permet à **_Philips_** à fournir des mises à jour de sécurité et de fonctionnalités de façon rapide, transparente, cohérente, fiable et sécurisée.

Procédure
=========

Le _**Système de mises à jour silencieuses**_ télécharge et prépare automatiquement les nouvelles versions de l'application Reacts en arrière plan. Dès que l'utilisateur fermera et rouvrira l'application Reacts, la nouvelle version se lancera. Ce système n'impacte pas l'utilisation de l’application : les mises à jour s'appliquent sans générer aucune interruption.

Fréquence de livraison
======================

Fastlane
--------

Par défaut, toute nouvelle installation de Reacts est souscrite à la _**FastLane**_. Les utilisateurs qui ont souscrit à la _**FastLane**_ recevront toutes les nouvelles mises à jour. Les utilisateurs qui ont souscrit à la _**Fastlane**_ bénéficieront des fonctionalités et des correctifs de sécurité les plus récents.
Les utilisateurs souscrivent à la _**FastLane**_ de façon automatique lors de l'installation regulière de l'application.

Documentation Technique
=======================

Noms de domaine
---------------

Pour permettre au système de fonctionner, les noms de domaines suivants doivent être ajoutés à la liste blanche dans les pare-feux organisationnels. Ils font partie des noms de domaine déjà utilisés par l'application Reacts.

 

Nom de domaine

But

`svc.iitreacts.com`

Le service se connectera régulièrement à cette API afin de déterminer si de nouvelles versions sont disponibles.

`www.iitreacts.com`

Si une nouvelle version est disponible, le service téléchargera le paquet à cette adresse.

Ports Sortants
--------------

Seul le port **443** est utilisé.

Ports Entrants
--------------

Aucun port entrant n'est utilisé.

Protocoles
----------

Seul HTTPS est utilisé.
