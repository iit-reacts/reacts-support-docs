title: Comment installer Reacts sans la fontionnalié de partage d'écran ou le de contrôle a distance
---
Si vous souhaitez installer Reacts sans fonctionnalités de partage d'écran ou de contrôle a distance, au cours du processus d'installation, lorsque la fenêtre vous permettant de choisir le type d'installation s’affiche, veuillez sélectionner la configuration personnalisée (**Custom**) :

![](@assets/attachments/Custom_install_1.png)

Dans la fenêtre suivante, ouvrez le menu déroulant de la fonctionnalité à désactiver et cliquez sur " This feature will not be available".  
Veuillez noter que la désactivation de la fonctionnalité de partage d'écran désactivera aussi la fonctionnalité de contrôle à distance.

![Windodws_-CustomInstalled01.png](@assets/attachments/Windodws_-CustomInstalled01.png)

Suivez l'assistant d'installation pour terminer l'installation:

![Windodws_-CustomInstalled02.png](@assets/attachments/Windodws_-CustomInstalled02.png)
