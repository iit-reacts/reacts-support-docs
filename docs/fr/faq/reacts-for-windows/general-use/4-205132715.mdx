**Équipement suggéré:**

  
**Portable PC** répondant aux exigences listés dans notre documentation sur les exigences système.

**Souris**

**Caméras Web HD**

**Bras articulé pour soutenir les caméras.** Nous commandons nos bras articulés du fournisseur B&H (New-York). Nous vous recommandons d’acheter quelques rallonges et pinces supplémentaires. Voici les détails:

2. **Pince supplémentaire**

![](@assets/attachments/madical_cart_3.png)

3\. **Rallonge**

![](@assets/attachments/medical_cart_4.png)

**Chariot Ergotron (Neo-Flex)**

![](@assets/attachments/medical_cart_5.png)

  
**Panier pour du stockage supplémentaire**

![](@assets/attachments/medical_cart_6.png)

**Microphone USB omnidirectionel** (pour une qualité audio supérieure, notamment dans les salles de conférence)

**Hauts parleurs portables (optionnel)** (pour une qualité audio supérieure à celle des haut-parleurs intégrés. Une variété de modèle est disponible sur le marché.)

**Convertisseur de signal vidéo numérique** (DVI/VHA /HDMI) **en signal USB** (pour les appareils d’échographie, les appareils à ultrasons, les moniteurs, les bronchoscopes, etc.). Le signal USB est automatiquement détecté par Reacts comme une entrée vidéo (n’oubliez pas d’installer le pilote du convertisseur) : Épiphan AV.iO HD

![Converter.png](@assets/attachments/Converter.png)

Vous pouvez vous référer à cette vidéo en anglais ([https://www.youtube.com/watch?v=aePGHV76M6I](https://www.youtube.com/watch?v=aePGHV76M6I)) expliquant comment utiliser un convertisseur de signal vidéo avec Reacts.

\*\* D’autres convertisseurs du même type sont offerts sur le marché

**Convertisseur de signal vidéo analogique en signal USB.** Le signal USB est automatiquement détecté par Reacts comme une entrée vidéo (n’oubliez pas d’installer le pilote du convertisseur). Plusieurs modèles sont offerts sur le marché et peuvent être trouvés sur internet.

  
**Autres accessoires:**

**Câble Ethernet** (3 à 6 mètres) si vous utilisez une connexion Internet filaire

**Rallonge USB avec amplificateur de signal (optionnelle)** – si un long câble USB est requis pour la caméra Web ou le convertisseur vidéo.
