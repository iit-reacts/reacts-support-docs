title: Comment réinitialiser mon mot de passe Reacts?
---
Vous pouvez modifier votre mot de passe dans Reacts en accédant à « Paramètres » --> « Mon compte » et en choisissant l’option « Réinitialiser le mot de passe »

![Reset_password_FR.png](@assets/attachments/Reset_password_FR.png)

Les directives pour la réinitialisation du mot de passe vous seront envoyées par courriel.

Vous pouvez également modifier votre mot de passe en visitant la page de connexion sur notre site internet : cliquez sur le bouton d'oubli de mot-de-passe, inscrivez l’adresse courriel que vous avez indiquée en créant votre compte et nous vous enverrons les directives pour modifier votre mot de passe.
