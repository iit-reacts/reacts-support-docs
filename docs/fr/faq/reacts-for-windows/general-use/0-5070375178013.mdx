title: Connexion sur Reacts Windows
---
À partir du 05 mai 2022, La façon de se connecter sur Reacts installé sur Windows a changé pour correspondre aux flux de connexion de nos autres plateformes afin de fournir aux utilisateurs une expérience unifiée.

Pour pouvoir vous connecter, vous devez :

- Ouvrir Reacts Windows;

- Cliquer sur "Se connecter";

![mceclip0.png](@assets/attachments/mceclip0.png)

Vous serez automatiquement redirigés vers notre site internet pour finaliser votre connexion (votre navigateur par défaut sera utilisé);

- Entrer vos identifiants de connexion (Adresse courriel et Mot de passe) puis cliquer sur "Me connecter";

![mceclip1.png](@assets/attachments/mceclip1.png)

- Cliquer sur "Ouvrir Reacts Loader" pour ouvrir Reacts installer sur Windows;

![mceclip9.png](@assets/attachments/mceclip9.png)

Reacts s'ouvrira et vous serez alors connectés à votre compte.
