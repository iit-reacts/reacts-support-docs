*   Identification des licences « pro » et « invité » dans le panneau de contacts

Pour vous permettre de mieux identifier vos contacts qui ont une licence de type « Pro » ou de type « Invité », nous avons ajouté un descriptif visuel dans le panneau de contacts. Les utilisateurs qui ont une licence de type «Standard » n’ont pas de descriptif visuel spécifique.Optimisation et correction de bogues
