title: Reacts iOS Comment mettre à jour l’application Reacts lorsqu’une nouvelle version obligatoire est disponible 
---
Prenez note qu’il est possible que votre appareil iOS fasse les mises à jour des applications automatiquement. Dans ce cas, vous pouvez continuer à utiliser Reacts sans interruptions.

Mais si cela n’est pas le cas, voici les étapes à suivre pour mettre à jour votre application:

1.  Ourvrir l’application Reacts
2.  Se connecter à l'application: une nouvelle fenêtre “Mise à jour requise” s’affichera
      
    
3.  Appuyer sur le bouton “Mettre à jour Reacts” :
      
    ![IMG_0157_edit.PNG](@assets/attachments/IMG_0157_edit.PNG)
      
    
4.  Vous serez redirigé vers la page du AppStore pour Reacts
      
    
5.  Appuyer sur le bouton “Mettre à jour”:
      
    ![IMG_0158_Edit.PNG](@assets/attachments/IMG_0158_Edit.PNG)
6.  Réouvrir Reacts
7.  La mise à jour est complétée et vous pouvez vous reconnecter.
