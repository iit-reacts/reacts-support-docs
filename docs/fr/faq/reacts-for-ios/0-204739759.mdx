title: Connexion Internet faible ou intermittente affectant le fonctionnement de Reacts mobile pour iOS
---
Une connexion Internet stable et fiable est requise pour un fonctionnement optimal de Reacts mobile sur iOS. Une connexion Internet faible ou instable pourrait affecter le fonctionnement de certaines fonctionnalités de base de Reacts, par exemple :

*   aucun flux vidéo de l’un des utilisateurs (ou des deux);
*   flux vidéo sans son;
*   impossible de partager des éléments avec votre interlocuteur;
*   incapacité à visionner un élément vidéo provenant de l’interlocuteur;
*   décalage des actions réalisées sur les éléments partagés.
