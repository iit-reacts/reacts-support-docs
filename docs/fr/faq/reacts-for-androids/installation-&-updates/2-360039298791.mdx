title: Reacts Android Comment mettre à jour l’application Reacts lorsqu’une nouvelle version obligatoire est disponible
---
Prenez note qu’il est possible que votre appareil Android fasse les mises à jour des applications automatiquement. Dans ce cas, vous pouvez continuer à utiliser Reacts sans interruptions.

Mais si cela n’est pas le cas, voici les étapes à suivre pour mettre à jour votre application:

1.  Ouvrir l’application Reacts
2.  Se connecter à l'application : une nouvelle fenêtre “Échecs de connexion” s’affichera
      
    
3.  Appuyer sur le bouton “Télécharger Reacts” :
      
    ![Screenshot_04_Edit_final_mix.jpg](@assets/attachments/Screenshot_04_Edit_final_mix.jpg)
      
    
4.  Vous serez redirigé vers la page du Google Play Store pour Reacts
      
    
5.  Appuyer sur le bouton “Mettre à jour” :
      
    ![Screenshot_03_Edit_final_mix.jpg](@assets/attachments/Screenshot_03_Edit_final_mix.jpg)
6.  Réouvrir Reacts
7.  La mise à jour est complétée et vous pouvez vous reconnecter.
