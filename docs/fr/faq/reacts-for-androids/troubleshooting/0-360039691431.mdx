title: Reacts me demande d'utiliser un appareil mobile chiffré et avec un PIN (mot de passe)
---
Afin d’assurer un environnement sécure durant l’utilisation de Reacts, nous imposons que les appareils sur lesquels vous désirez utiliser Reacts soient munis d’un mot de passe (ou NIP) et chiffrés.

Si cela n’est pas le cas, vous allez rencontrer le message suivant lorsque vous accdéderez à Reacts:

![Security_Issue_FR_HV.jpg](@assets/attachments/Security_Issue_FR_HV.jpg)

**Voici** **les** **étapes** **à** **suivre** **pour** **ajouter** **un mot de** **passe** **ou** **NIP**:

_Il_ _est_ _possible que les_ _étapes_ _suivantes_ _ne soient pas les_ _mêmes_ _sur_ _tous_ _les_ _appareils__._

1.  Ouvrir les Paramètres de votre appareil. Cela peut être accompli soit par:
    1.  Glisser votre doigt sur le haut de l’écran vers le bas et appuyer sur le symbole des paramètres dans le menu qui s’ouvre.
    2.  Ouvrir la liste des applications et appuyer sur “Paramètres”.
2.  Ouvrir la section “Sécurité”
3.  Ouvrir la section “Écran de verrouillage”
4.  Vous pouvez ici mettre en place un mot de passe ou un NIP pour votre appareil.

**Voici** **les** **étapes** **à** **suivre** **pour chiffrer votre appareil** (_prenez_ _note que_ _depuis_ _Android 8, les_ _appareils_ _sont_ _chiffrés_ _par_ _défaut__)_ : 

_Il_ _est_ _possible que les_ _étapes_ _suivantes_ _ne_ _soient_ _pas les mêmes sur tous les appareils._ 

1.  Ouvrir les Paramètres de votre appareil. Cela peut etre accomplis soit par: 
    1.  Glisser votre doigt sur le haut de l’écran vers le bas et appuyer sur le symbole des paramètres. 
    2.  Ouvrir la liste des applications et appuyer sur “Paramètres”. 
2.  Ouvrir la section “Sécurité” 
3.  Ouvrir la section “Options avancée” 
4.  Ouvrir la section "Chiffrement”. 
5.  Vous pouvez ici chiffrer votre appareil.
