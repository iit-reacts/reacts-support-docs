Renouvellement obligatoire de mon mot de passe
==============================================

Pour des raisons de sécurité, tout mot de passe doit être changé au minimum une fois par an.

Mot de passe arrivant à expiration
----------------------------------

Si vous n'avez pas changé votre mot de passe depuis 358 jours (un an moins 7 jours), vous recevrez un courriel vous indiquant que votre mot de passe va expirer. Il vous sera demandé de le réinitialiser.

Vous un exemple de courriel reçu :

![S04049_01.png](@assets/attachments/S04049_01.png)

Pour effectuer cette réinitialisation:  
- Cliquer sur le bouton orange "Réinitialiser le mot de passe" :vous serez alors redirigé vers notre site internet

- Connectez-vous avec votre compte Reacts

- Suivez les instructions pour changer votre mot de passe

Mot de passe expiré
-------------------

Si vous n'avez pas changé votre mot de passe depuis 365 jours, votre mot de passe est alors expiré.

Pour réinitialiser votre mot de passe, veuillez :

- Cliquer sur le bouton orange "Réinitiliser le mot de passe" de votre courriel "Votre mot de passe Reacts expirera dans 7 jours" : vous serez alors redirigé vers notre site internet

- Suivez les instructions pour changer votre mot de passe

Une tentative de connexion sur la plateforme Reacts avec un mot de passe expiré vous affichera le message suivant :

![S04049_05.png](@assets/attachments/S04049_05.png)

Vous pourrez alors cliquer sur "Reset Password" et suivre les instructions afin de le réinitialiser.

Note importante
----------------

Pour des raisons de sécurité, il n'est pas possible de réutiliser votre précédent mot de passe lors d'un changement de mot de passe.

Si une telle action est effectuée, vous recevrez un message d'erreur vous informant de la non-possibilité de réutiliser le même mot de passe (comme ci-dessous).
