title: Comment changer les hauts-parleurs utilisés par la WebApp
---
Reacts WebApp utilise les hauts-parleurs sélectionnés par défaut dans les paramêtres de votre ordinateur.

Si vous souhaitez utiliser d'autres hauts-parleurs, you devez :

- ouvrir les paramètres de votre ordinateurs

- aller dans les paramètres de son

- Sélectionner les hauts-parleurs que vous souhaitez utiliser dans la WebApp

- Vous connecter à Reacts WebApp
