title: Pourquoi est-ce que je reçois des courriels Nouveau message
---
Vous recevrez un courriel Reacts si vous recevez un message dans Reacts alors que vous n'êtes pas connecté à l'application.

Veuillez noter que vous ne recevrez pas un courriel pour chaque message reçu. Vous recevrez seulement un courriel par conversation par 30 minutes.

Par exemple : si votre interlocuteur vous envoie plusieurs messages au courant des 50 prochaines minutes alors que vous n'êtes pas connectés à Reacts, vous recevrez 2 courriels de notifications.
