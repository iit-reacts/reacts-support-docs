### Requis

Lorsqu'un compte Reacts est créé ou lorsque un mot de passe est réinitialisé/changé:

le mot de passe doit contenir au minimum 8 caractères dont un des caractères doit être une lettre MAJUSCULE ainsi qu'avoir un caractère de type numérique.

### Expiration de mot de passes

Le mot de passe d'un compte Reacts doit être changé au minimum une fois par an. Si ce n'est pas le cas, le mot de passe Reacts expirera.

L'utilisateur de ce compte recevra une notice 7 jours avant l'expiration de celui-ci et il lui sera demandé de changer son mot de passe avant l'expiration.

L'utilisateur sera forcé de changer de mot de passe pour pouvoir se connecter à nouveau si le mot de passe n'est pas changé dans les 7 jours suivant cette notice.

### Ré-utilisation de mot de passes

Lors du changement ou de la réinitialisation du mot de passe Reacts, l'utilisateur ne peut pas ré-utiliser le même mot de passe que celui actuellement utilisé.

**Note : Ces paramètres ne peuvent pas être modifiés par les utilsateurs ou les administrateurs d'abonnements Reacts**
