**Comment modifier vos paramètres de langue?**

Lorsque vous ouvrez une session Reacts pour la première fois, votre langue par défaut sera la même que celle que vous avez choisie lors de votre inscription. Pour changer votre langue par défaut, sélectionnez l’option **Paramètres de langue**, choisissez la langue de votre choix et confirmez votre sélection.

Note : Reacts mobile utilisera la même langue que votre appareil. Si la langue de votre appareil n'est pas disponible dans Reacts, Reacts sera en anglais.

**Comment modifier vos paramètres de réseau?**

Configurez vos paramètres de réseau et de serveur au besoin (en fonction du niveau de sécurité de l’emplacement depuis lequel vous vous connectez)

![parametre_reseau_sans_nombre.png](@assets/attachments/parametre_reseau_sans_nombre.png)

**Vous pouvez consulter le tutoriel vidéo [Paramètres du réseau (version anglaise)(](https://www.youtube.com/watch?v=4Nj9aY2InJU)**

Note: Reacts WebApp utilisera les mêmes paramètres que ceux présents dans les paramètres de Chrome.

Reacts mobile utilisera les mêmes paramètres que ceux présents dans les paramètres de votre téléphones.

**Comment récupérer vos entrées supprimées?**

Sélectionnez l’entrée dont vous avez besoin dans la liste déroulante et sélectionnez l’option **Réactiver**.

![](@assets/attachments/settings5.png)

**Vous pouvez consulter le tutoriel vidéo [Entrées supprimées (version anglaise)(](https://www.youtube.com/watch?v=LkHJjtOrZqg)**

Note: Les fonctionnalités de suppression et reactivation d'entrées sont uniquement disponibles sur Reacts Windows.
