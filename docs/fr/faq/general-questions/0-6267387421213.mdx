title: Dernière version de Reacts VS Version minimale prise en charge
---
Pour utiliser Reacts, nous vous recommandons fortement de toujours avoir la dernière version disponible.

Vous trouverez dans le tableau ci-dessous la dernière version pour chaque plateforme et la version minimale supportée.

Si vous avez une version plus ancienne que la version minimale prise en charge, vous ne pourrez pas vous connecter et devrez installer la dernière version.

Note :

Reacts sur Windows, iOS et Android se mettra à jour automatiquement SAUF si vous désactivez la mise à jour automatique (ce que nous déconseillons).

Reacts WebApp sur Chrome se mettra automatiquement à jour.
