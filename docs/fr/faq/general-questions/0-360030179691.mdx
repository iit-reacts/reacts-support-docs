title: Comment paramétrer l'authentification à deux facteurs
---
En plus de votre mot de passe Reacts, vous pouvez ajouter un second paramètre de sécurité à votre compte Reacts.

Pour cela, veuillez suivre les étapes suivantes :

- Connectez-vous à Reacts puis allez sur votre page "Mon compte" (pour plus d'information veuillez consulter notre page "Comment accéder à la section "Mon Compte" depuis la WebApp")

- À la page "Mon compte" : cliquer sur le bouton "Configurer l'authentification à deux facteurs"
  

![authentification_double_facteur_01.jpg](@assets/attachments/authentification_double_facteur_01.jpg)

- Pour des raisons de sécurité, il vous est demandé de vous reconnecter : entrer vos informations et cliquer sur "Me connecter"

![compte_06.jpg](@assets/attachments/compte_06.jpg)

- Suivre les étapes indiquées dans la page "Configurer l'authentification à deux facteurs"

- Entrer le code d'authentification indiqué sur votre application

- Cliquer sur le bouton "Valider"

![authentification_double_facteur_02.jpg](@assets/attachments/authentification_double_facteur_02.jpg)

Un message de validation vous indiquera que votre double facteur est alors configuré :

![authentification_double_facteur_03.jpg](@assets/attachments/authentification_double_facteur_03.jpg)

  
Dès votre prochaine connexion, sur n'importe quelle plateforme Reacts, après avoir entré votre adresse courriel, votre mot de passe et cliqué sur le bouton "Connexion", il vous sera demandé d'indiquer votre code de double authentification :

![web2FA_pageFR.jpg](@assets/attachments/web2FA_pageFR.jpg)

**NOTE : si vous avez besoin de réinitialiser ou supprimer votre authentification à double facteurs, veuillez demander à votre institution de contacter le support de Reacts**
