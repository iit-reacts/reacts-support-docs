APERÇU
------

Votre abonnement PRO vous permet de donner des abonnements Reacts gratuits aux personnes que vous souhaitez.

Cette abonnement :

- donne un accès gratuit à Reacts

- vous met directement en contact dans votre liste de contact Reacts

- vous permet de communiquer par appel ou messagerie sécurisée.

Veuillez noter que vous pourrez leur retirer l'accès à tout moment et l'invité sera retiré de votre liste de contact.

PRÉREQUIS
---------

Votre compte Reacts doit avoir un abonnement PRO afin de pouvoir accéder à cette fonctionnalité.

Vous devez dans un premier temps accéder au tableau de bord Reacts.

Une fois connecté, veuillez sélectionner l'Organisation dans laquelle vous possédez l'abonnement PRO :

![assign01.png](@assets/attachments/assign01.png)Si vous n'avez accès qu'à une organisation, cette étape sera sautée.

COMMENT ASSIGNER UN ABONNEMENT INVITÉ
-------------------------------------

Assurez-vous d'être dans la section "Mes invités". Veuillez noter que si vous possédez des droits d'administrateur dans cette organisation, vous ne serez pas de façon automatique dans " Mes Invités".

![Step_2.png](@assets/attachments/Step_2.png)

La liste de vos invités est listée ici:

![Steps_2_bis.png](@assets/attachments/Steps_2_bis.png)

Cliquer sur "Ajouter un invité"

![Step_3.png](@assets/attachments/Step_3.png)

Renseigner l'adresse courriel, le prénom et le nom de la personne à qui vous souhaitez donner un abonnement Invité.

![assign05.png](@assets/attachments/assign05.png)

Vous pouvez cliquer sur l'icône + ou sur "+import CSV" (voir la section " COMMENT ASSIGNER DES ABONNEMENTS INVITÉS AVEC UN FICHIER CSV ci dessous pour plus d'information) pour assigner plusieurs abonnements Invités à la fois.

![assign06.png](@assets/attachments/assign06.png)![assign07.png](@assets/attachments/assign07.png)

Cliquer sur "Ajouter" pour assigner les abonnements

![assign08.png](@assets/attachments/assign08.png)

**Note importante : Assurez-vous que les personnes ajoutées à votre liste d'invités ont bien accepté l'abonnement en se connectant à leur compte Reacts. Si l'abonnement n'est pas accepté, vous ne serez pas en mesure de communiquer ensemble via Reacts.**

COMMENT ASSIGNER DES ABONNEMENTS INVITÉS AVEC UN FICHIER CSV
------------------------------------------------------------

Notre fonctionnalité d'import de fichier au format CSV vous permet d'assigner plusieurs abonnements à la fois.

Vous pourrez trouver ci-dessous les informations à renseigner dans votre fichier pour que l'import puisse être fait.

### Créer un document CSV

Pour créer un document au format CSV, veuillez :

- ouvrir un nouveau fichier excel

- le sauvegarder en sélectionnant le format "CSV UTF-8"

### Format des informations à indiquer dans votre CSV

Votre document CSV doit contenir les informations suivantes :

*   Colonne A  de votre CSV :
    *   Ligne 1 : « Prénom »
    *   Ligne 2 : le prénom de la personne à qui assigner l’abonnement

*   Colonne B de votre CSV :
    *   Ligne 1 : « Nom »
    *   Ligne 2 : le nom de la personne à qui assigner l’abonnement

*   Colonne C de votre CSV :
    *   Ligne 1 : « Adresse courriel »
    *   Ligne 2 : le courriel de la personne à qui assigner l’abonnement

Ajouter ensuite une nouvelle ligne pour chaque abonnement que vous souhaitez assigner.

**Note importante** : L'import de votre CSV fonctionnera uniquement si vous respectez le format détaillé ci- dessus.

COMMENT DÉSASSIGNER UN ABONNEMENT INVITÉ
----------------------------------------

Assurez-vous d'être dans la section "Mes invités". Veuillez noter que si vous possédez des droits d'administrateur dans cette organisation, vous ne serez pas automatiquement dans la section " Mes Invités".

![Steps_2_bis.png](@assets/attachments/Steps_2_bis.png)

Cliquer sur le bouton "RETIRER L'INVITÉ" qui correspond à l'invité à qui désassigner l'abonnement

![Step_8_EN.png](@assets/attachments/Step_8_EN.png)

Cliquer sur le bouton "SUPPRIMER" de la demande de confirmation

![Step_9_EN.png](@assets/attachments/Step_9_EN.png)

Votre invité n'est plus dans votre liste d'invité dans le tableau de bord, ni dans votre liste de contact Reacts
