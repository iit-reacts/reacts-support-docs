---
APERÇU
------

En tant qu'administrateur ou détenteur d'un abonnement PRO, vous pouvez accéder à votre tableau de bord Reacts pour gérer vos abonnements.  
NOTE: les administrateurs Reacts n'ont pas besoin d'avoir un abonnement Reacts valide pour gérer les abonnements. Les administrateurs ont seulement besoin d'avoir un compte Reacts.

ÉTAPES À SUIVRE SI VOUS UTILISEZ LE BOUTON "ACCEDER AU TABLEAU DE BORD"
-----------------------------------------------------------------------

- Aller sur notre site internet

- Cliquer sur le bouton "Aller au tableau de bord"

![Website_FR.png](@assets/attachments/Website_FR.png)

ÉTAPES À SUIVRE SI VOUS ÊTES CONNECTÉS À LA WEBAPP
--------------------------------------------------

-  Cliquer sur le bouton "tableau de bord".

Ce bouton est disponible au centre de la page principale et à tout moment dans le menu de gauche.

![tableau_de_bord_depuis_webapp.png](@assets/attachments/tableau_de_bord_depuis_webapp.png)

ÉTAPES À SUIVRE SI VOUS ÊTES SUR LA PAGE "MON COMPTE"
-----------------------------------------------------

- À la page "Mon compte" :

1.  Cliquer sur le bouton "Gestion d'équipes" pour gérer vos abonnements et vos équipes
2.  Cliquer sur le bouton "Gestion des invités" pour assigner des abonnements invités"
