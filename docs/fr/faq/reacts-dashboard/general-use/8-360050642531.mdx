APERÇU
=========

Ce tutoriel vous expliquera comment ajouter des droits d'administrateurs.

PRÉREQUIS
=========

*   Vous devez dans un premier temps accéder au tableau de bord Reacts. Vous pourrez trouver les étapes pour y accéder dans le tutoriel suivant : [Comment accéder à mon tableau de bord]({{ site.baseurl }}{% link fr/faq/reacts-dashboard/general-use/0-360050582152.md %})
*   Vous devez possèder des droits d'administrateur dans l'organisation (ou unité d'organisation) où vous souhaitez ajouter un administrateur

ÉTAPES À SUIVRE POUR AJOUTER DES DROITS D'ADMINISTRATEUR
========================================================

1.  ### Choisissez l'onglet "Permissions"![1-_choose_permissions.png](@assets/attachments/1-_choose_permissions.png)
    
2.  ### Cliquez sur le bouton orange "ajouter un administrateur"![2-_add_admin.png](@assets/attachments/2-_add_admin.png)
    
3.  ### Cliquez sur les petites flèches afin de voir toutes vos unités d'organisation disponibles![3-_open_dropdown_menus.png](@assets/attachments/3-_open_dropdown_menus.png)
    
4.  ### Sélectionnez à quel niveau vous désirez ajouter les droits d'administrateur, puis cliquez sur "sauvegarder"
    

_Vous pouvez avoir plus de détail entre les deux types de droits en cliquant sur le point d'interrogation situé à coté de "Type de droit d'administrateur"![4-_choose_lvl_and_where.png](@assets/attachments/4-_choose_lvl_and_where.png)_

### 5\. Lorsque vous verrez une confirmation au haut de l'écran, vous pourrez appuyer sur "fermer"![5-_confirmation__fermer.png](@assets/attachments/5-_confirmation__fermer.png)

ÉTAPES À SUIVRE POUR RETIRER DES DROITS D'ADMINISTRATEUR
========================================================

1.  ### Choisissez l'onglet "Permissions"![1-_choose_permissions.png](@assets/attachments/1-_choose_permissions.png)
    
2.  ### Cliquez sur "Ouvrir" au bout de la ligne de l'administrateur que vous souhaitez modifier![2-_open_admin.png](@assets/attachments/2-_open_admin.png)
    
3.  ### Cliquez sur les petites flèches afin de voir tous vos unités d'organisation disponible![3-_open_dropdown_menus_remove_.png](@assets/attachments/3-_open_dropdown_menus_remove_.png)
    
4.  ### Décochez toutes les cases pour retirer complètements les droits d'administrateurs, puis appuyez sur "Sauvegarder"![4-_uncheck_all_boxes.png](@assets/attachments/4-_uncheck_all_boxes.png)
    
5.  ### Lorsque vous verrez une confirmation au haut de l'écran, vous pourrez appuyer sur "fermer"![5-_confirmation__fermer_remove_.png](@assets/attachments/5-_confirmation__fermer_remove_.png)
