APERÇU
------

En tant qu'administrateur vous pouvez distribuer des abonnements non assignés d'une Unité d'Organisation à une autre.

**\*Notez que ce tutoriel est seulement pour déplacer des abonnements non-assignés\***

**\*Si vous souhaitez déplacer des utilisateurs ainsi que leurs abonnements, voir le tutoriel : Comment déplacer vos utilisateurs d'une unitée d'organisation vers une autre**

PRÉREQUIS
---------

Vous devez être administrateur d'au minimum deux Unités d'Organisation et avoir des abonnements non assignés disponibles.

Vous devez dans un premier temps accéder au tableau de bord Reacts.

DISTRIBUER DES ABONNEMENTS
--------------------------

### 1\. Cliquer sur "Distribuer des abonnements"

![-distribute01.png](@assets/attachments/-distribute01.png)

Vous verrez alors :

- L'Unité d'organisation source : l'Unité qui détient les abonnements que vous allez prendre. Cette Unité d'Organisation peut être changée en cliquant sur son nom.

- Le nombre d'abonnements non assignés disponibles dans cette unité

![-distribute02.png](@assets/attachments/-distribute02.png)

### 2\. Cliquer sur "Sélectionner une Unité d'Organisation de destination"

![-distribute03.png](@assets/attachments/-distribute03.png)

### 3\. Sélectionner l'Unité dans laquelle vous voulez mettre vos abonnements

### 4\. Cliquer sur "Sélectionner cette Unité d'Organisation"

![-distribute04.png](@assets/attachments/-distribute04.png)

### 5\. Indiquer le nombre d'Abonnement PRO et/ou Standard que vous voulez transférer

### 6\. Cliquer sur "Appliquer"

![-distribute05.png](@assets/attachments/-distribute05.png)

Note : vous pouvez effectuer plusieurs distributions à la fois en cliquant sur le bouton "+" situé en bas à gauche :

![-distribute06.png](@assets/attachments/-distribute06.png)
