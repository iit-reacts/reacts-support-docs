Nouvelles fonctionnalités :

- Améliorations IU / XU

- Les utilisateurs Pro peuvent maintenant télécharger un fichier CSV avec la liste de leurs invités

- Retirer les droits administrateur à un utilisateur, le retirera aussi de la liste des administrateurs

- Nouvelle colonne pour savoir si une machine est expirée ou non-active

- Les administrateurs peuvent maintenant renommé les machines

Optimisation générale et correction de bogues.
