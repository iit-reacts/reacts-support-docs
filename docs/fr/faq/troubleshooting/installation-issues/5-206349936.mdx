title: Je ne peux pas installer Reacts, le message une autre installation est en cours s'affiche.
---
Vous êtes en train d'installer Reacts et ce message s'affiche : "une autre installation est en cours. Vous devez terminer cette installation avant de poursuivre."

![2017-12-29_14-20-13.png](@assets/attachments/2017-12-29_14-20-13.png)

**S'assurer qu'aucune autre installation n'est en cours ou en attente en arrière plan**

Cliquer sur le bouton "Annuler" du message ci-dessus, ouvrir le Gestionnaire des tâches --> Onglet processus, rechercher des processus "msiexec" en cours ou en fin d'exécution.
