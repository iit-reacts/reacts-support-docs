title: Web app gelée, ne répondant pas
---
Dans le but d'optimiser votre expérience de navigation sur internet et la rapidité d'action et de conserver les bandes passantes de communications, votre navigateur internet met en cache (sauvegarde) certaines de vos informations web.

**Veuillez noter qu'AUCUNE des informations de votre compte Reacts ou de vos communications n'ait sauvegardé.**

Malheureusement, parfois, après une mise à jour de votre application web, le cache peut causer des problèmes inattendus (par exemple : absence de certaines informations, incapacité d'effectuer certaines fonctionnalités de la Web App).

Si vous rencontrez ce comportement, veuillez s'il vous plaît:

1.  Vous connecter sur la Web app de Reacts
2.  Une fois sur la page principale, appuyer sur CTRL+SHIFT+R (en même temps)
3.  La Web App sera rechargée automatiquement

Le raccourci CTRL+SHIFT+R recharge la page en ignorant le contenu du cache.
