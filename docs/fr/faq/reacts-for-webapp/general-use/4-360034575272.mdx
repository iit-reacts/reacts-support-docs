title: Comment changer les permissions de mes notifications, ma caméra et de mon micro sur la Reacts web app
---
Lors de votre première connexion à la Reacts web app sur votre navigateur Chrome, il vous a été demandé d'accepter ou de refuser recevoir des notifications, ainsi que d'accepter ou de refuser l'utilisation de votre caméra et de votre micro dans Reacts web app.

Cette acceptation ou ce refus sera appliqué lors de vos prochaines connexions à la WebApp sur votre navigateur.

Comment changer les permissions ?
=================================

Pour changer vos permissions, veuillez suivre les informations suivantes :

1.  Se connecter sur la WebApp
2.  Cliquer sur le cadenas se situant à gauche de la barre d'adresse de Chrome
3.  Changer les permissions
4.  Fermer la fenêtre des permissions
5.  Rafraîchir la page (tel que demandé par Chrome) afin d'appliquer les changements
