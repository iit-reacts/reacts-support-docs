title: Comment accéder à la section Mon Compte depuis la WebApp
---
Lorsque vous êtes connectés sur la WebApp, vous pouvez à tout moment accéder aux informations concernant votre compte sur notre site internet.

Pour cela, veuillez :

- Cliquer sur les trois buttons à côté de votre nom (en haut à gauche)

![compte_01.jpg](@assets/attachments/compte_01.jpg)

- Cliquer sur "Retourner au site web" : une nouvelle page internet s'ouvrira automatiquement
  
![compte_02.jpg](@assets/attachments/compte_02.jpg)

- Cliquer sur la flèche à côté de votre nom (en haut à droite)
  
![compte_03.jpg](@assets/attachments/compte_03.jpg)

-Cliquer sur "Mon compte"
  
![compte_04.jpg](@assets/attachments/compte_04.jpg)

  
Vous pourrez alors consulter les différentes informations de votre compte ainsi que configurer l'authentification à deux facteurs
