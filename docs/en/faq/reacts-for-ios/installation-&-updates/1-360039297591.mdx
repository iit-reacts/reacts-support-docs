Please note that it is possible that application updates are done automatically onyouriOS device. In which case, you can continue to use Reacts without interruptions.

If that is not the case, here are the steps to update your application manually:

1.  Open Reacts  
      
    
2.  Login to Reacts: a new “Update required” window will open
    
3.  Tap on the “Update Reacts” button :  
      
    ![IMG_0160_edit.PNG](@assets/attachments/IMG_0160_edit.PNG)
      
    
4.  You will be redirected to Reacts’ App Store page
5.  Tap on the “Update” button:  ![IMG_0161_Edit.PNG](@assets/attachments/IMG_0161_Edit.PNG)
6.  Reopen Reacts  
      
    
7.  The update is complete, and you can proceed to log in.
