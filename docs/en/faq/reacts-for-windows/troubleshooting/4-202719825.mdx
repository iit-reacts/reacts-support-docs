title: I receive Windows Firewall pop up message when using some of Reacts features for the first time
---
When using certain features of Reacts for the first time (example: share app/desktop) Windows Firewall will block them and the below message will pop up on your screen:

![](@assets/attachments/windows_firewall_message.png)

Simply choose _**Allow access**_ to continue
