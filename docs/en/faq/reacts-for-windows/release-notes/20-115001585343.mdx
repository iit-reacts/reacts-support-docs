title: Version 2.5, released April 4, 2017

*   Fixed a Memory leak problem when running from RDP
*   Added IPV6 Support
*   Performance optimization and stabilization
*   Files in the Reacts library and instant messaging can now be opened directly in Reacts
*   Bug fixing
*   Prioritization of UDP over TCP when UDP is allowed (1), resulting in a more optimal streaming experience.
*   Optimization of peer-to-peer (end to end) security using DTLS-SRTP in replacement of the previous ZRTP and tunneled connection (2).

(1)   IMPORTANT: This change is only pertinent to our streaming services (see highlighted UDP). All the other services remain unchanged.
