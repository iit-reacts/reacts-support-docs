##### Start a new session or open one of your previously created sessions

![menu_session_en.png](@assets/attachments/menu_session_en.png)

![](@assets/attachments/Session2.png)

**New session**- opens Main working area from where you can start creating a new session or connect with one of your contacts.

![](@assets/attachments/Session3.png)

**Saved Sessions**- allows you to access, manage or edit previously saved sessions.

**Scheduled Sessions** - allows you to schedule a session (acting as a host) or join a session with one or more of Reacts contacts (as attendee).

To learn more about Scheduled Session, please see :[How to schedule an event (https://www.youtube.com/watch?v=H8UOyQCswBk)](https://youtu.be/H8UOyQCswBk)
