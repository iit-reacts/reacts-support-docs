If you are connecting to a network that has network proxy set up, please follow the instructions below to configure proxy settings in Reacts:

At Reacts login screen choose option network settings:

![07.png](@assets/attachments/07.png)

At Network settings windows choose option manual and add the settings:

![](@assets/attachments/mceclip0.png)

**Important note**

In order to use our silent updates feature you need to additionally configure proxy settings. If needed, please follow the steps in our documentation "How to set up proxy settings for Reacts Silent update feature".
