title: How to change the speaker to use on the WebApp
---
Reacts Webapp use the speakers selected as the default ones in the Setting of your PC.

If you want to use other speakers in the WebApp you will need to :

- open the settings for your PC
- go to the sounds settings

- Put the speaker you want to use in the WebApp as the default ones

- log in in Reacts WebApp
