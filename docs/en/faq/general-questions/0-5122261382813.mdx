### Password requirement

When creating a new Reacts account or resetting / changing the password of an existing Reacts account :

The password must be at least 8 characters including at least one uppercase letter, one lowercase letter and one numeric character.

### Password expiration

A Reacts account's password need to be changed at least once a year. If it's not changed, the password will expire.

The user will receive a disclaimer 7 days before the expiration of the password and will be asked to change the password of the account

The user will be forced to change the password in order to login again if it's not done within the 7 days following the disclaimer.

### Password re-use

When changing or resetting a Reacts password, the user cannot use the same password as the one currently used.

**Note : those settings cannot be modified by users or Reacts subscriptions' administrators**
