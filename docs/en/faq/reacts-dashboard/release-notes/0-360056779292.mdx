title: Version 3.9.2, released Feb 20, 2021
---
New functionalities:

- UI / UX improvements

- Pro users can now download a CSV of their own guests

- Removing admin rights to a user will remove the user from your list of admins

- New column in the machine tab to see if the machine is expired or non-active

- Administrators can now rename their machine(s)

General optimization and bug fixes.
